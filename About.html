About

  • Evolution is an abstract universal phenomena that exists in many fields. Visualization and visual analysis technology is naturally suitable for analyzing evolutionary phenomena. We manually collected more than 110 articles about the Evolution visualization and visual analytics, and proposed evolutionViz to visualize publications' information. The website(http://evo.regvis.net) provides an intuitive visual interface for the display and classification of article information, and users can perform multi-criteria filter exploration and fixed text search.The visual interface will be updated with user interaction.
